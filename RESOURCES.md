# Resources — one place, curated hard

Rule of thumb: at any moment you should be working from at most ONE book,
ONE video series, and ONE practice site. Hoarding tabs is procrastination
wearing a productivity costume.

## Python (module 01)
- Official tutorial (ch. 3–5, 9): https://docs.python.org/3/tutorial/
- *Automate the Boring Stuff*, free: https://automatetheboringstuff.com/
- Exercism Python track (free, human mentors): https://exercism.org/tracks/python
- When you're past basics: *Fluent Python* (Ramalho) — the "second book," not the first

## Agents & the Claude API (module 02)
- **Anthropic, *Building effective agents***: https://www.anthropic.com/engineering/building-effective-agents ← the one to read twice
- Tool use: https://docs.claude.com/en/docs/agents-and-tools/tool-use/overview
- Messages API: https://docs.claude.com/en/api/messages
- Anthropic cookbook (runnable notebooks): https://github.com/anthropics/anthropic-cookbook
- MCP spec & docs (you use it daily; eventually read how it works): https://modelcontextprotocol.io/

## Evals & observability (module 03)
- Hamel Husain, *Your AI Product Needs Evals*: https://hamel.dev/blog/posts/evals/
- Eugene Yan, LLM patterns: https://eugeneyan.com/writing/llm-patterns/
- Anthropic on success criteria & evals: https://docs.claude.com/en/docs/test-and-evaluate/define-success
- Braintrust docs (you have the account wired already): https://www.braintrust.dev/docs
- Wilson interval (small-n statistics): https://en.wikipedia.org/wiki/Binomial_proportion_confidence_interval

## Software design (module 04)
- Ousterhout, *A Philosophy of Software Design* — buy it, it's short
- Kleppmann, *DDIA* — ch. 1, 2, 5, 8
- pytest: https://docs.pytest.org/en/stable/getting-started.html
- FastAPI: https://fastapi.tiangolo.com/
- Microservices patterns: https://microservices.io/patterns/ (+ search "Fowler monolith first" for the antidote)
- refactoring.guru (patterns & smells, visual): https://refactoring.guru/

## ML & fine-tuning (module 05)
- 3Blue1Brown linear algebra + neural nets: https://www.3blue1brown.com/
- Karpathy, *Zero to Hero*: https://karpathy.ai/zero-to-hero.html ← the rite of passage
- *Mathematics for ML* (free book): https://mml-book.github.io/
- Hugging Face LLM course: https://huggingface.co/learn/llm-course
- Unsloth fine-tuning notebooks (free GPU via Colab): https://docs.unsloth.ai/
- Andrew Ng, ML Specialization (if you want a structured second pass): https://www.coursera.org/specializations/machine-learning-introduction

## RAG & knowledge assistants (module 08)
- Anthropic docs on RAG/embeddings + cookbook RAG notebooks: https://github.com/anthropics/anthropic-cookbook
- Full-cycle RAG evaluation (Hamel again): https://hamel.dev/blog/posts/evals/
- When you outgrow numpy-cosine: pgvector (Postgres extension) — https://github.com/pgvector/pgvector

## Web apps & databases (module 09)
- SQL, interactive: https://sqlbolt.com/ ← do this before any ORM
- FastAPI tutorial (whole thing is good): https://fastapi.tiangolo.com/tutorial/
- MDN Learn (HTML/CSS/JS canon): https://developer.mozilla.org/en-US/docs/Learn
- Chart.js: https://www.chartjs.org/
- Mermaid (diagrams-as-code for module 04's UML too): https://mermaid.js.org/

## Org transformation & the job (module 10 / P1)
- Anthropic, *Building effective agents* — re-read it wearing the work hat
- BJ Fogg-style pilot thinking is overkill; the pilot-one-pager template IS the resource
- HIPAA plain-language summary (know the shape, defer specifics to work counsel): https://www.hhs.gov/hipaa/for-professionals/privacy/index.html
